# Project Name
Dealership Management System

# Project Description
A basic Spring Boot project that simulates a dealership management system with vehicles, clients and sellers. I have built it to practice with Spring technologies.

# Project Features
- Add new clients, sellers or vehicles to the database
- Search functions
- Sell vehicles to clients
- Bootstrap has been used for styling
- Thymeleaf as templating engine

# Prerequisites
- Java 11
- Spring

# Usage
Open the project in IntelliJ and start it. <br>
The application can be accessed through the localhost: http://localhost:8080/ <br>
It uses in memory H2 database so all the data will be reseted once the application is restarted
